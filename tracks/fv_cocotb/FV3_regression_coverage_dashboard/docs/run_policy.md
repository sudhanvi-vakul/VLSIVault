# Run Policy — Project 3

## Regression goals
- Start small: 2 tests × 3 seeds (bring-up)
- Baseline: 10 seeds per test (daily)
- Closure: 50–100 seeds per test (before you claim sign-off)

## Artifact policy
- Always store: sim.log, metadata.json, trace.jsonl, coverage.json
- Store waves:
  - FAIL runs: always
  - PASS runs: only one sampled seed per test (documentation)

## Reproducibility rule
- Every run folder must include a replay command.
